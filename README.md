# 🎮 Animeverse Gaming 🕹️

## **Project Showcase**

**Animeverse Gaming** is a fully responsive, front-end web application that serves as a **single-page instant gaming portal**. It showcases modern web development skills by dynamically rendering multiple classic mini-games directly within a modal, eliminating the need for external downloads or page reloads.

This project was developed as a final year/semester project and serves as a comprehensive demonstration of expertise in **HTML5, CSS3, and modern JavaScript (ES6+)**.

-----

## 🌟 Features

  * **8 Classic Mini-Games:** Includes games like **Snake**, **Tic Tac Toe** (Player vs. Simple AI), **2048**, **Memory Match**, **Rock Paper Scissors**, **Color Match**, **Number Guesser**, and **Reaction Test**.
  * **Fully Responsive Design:** The entire website and all games are optimized for desktop, tablet, and mobile screens.
  * **Dynamic Game Loading (Modal):** Games are loaded and initialized dynamically in a modal using JavaScript, providing a smooth, fast, and continuous user experience.
  * **Modern CSS Styling:** Utilizes a custom, dark-themed "Gaming" color palette with neon and glow effects (`style.css`).
  * **Game Filtering:** Users can filter games by categories like **Arcade**, **Puzzle**, and **Strategy**.
  * **High Score Tracking:** Uses **Local Storage** to save high scores for games like Snake and 2048.
  * **Contact Form Integration:** Implements a functional contact form using **EmailJS** for real-time message delivery (requires user configuration).

-----

## 🛠️ Tech Stack

  * **Frontend:**

## 📁 Project Structure

The repository is organized logically to separate the core web files from the game-specific logic.

````
Animeverse-Gaming/
├── index.html        # Main structure and content (Home, Games, About, Contact sections)
├── style.css         # All custom CSS styling and responsiveness
├── games.js          # Main game data array and all individual game creation/initialization functions
├── script.js         # EmailJS configuration and Contact Form submission logic
└── OIP.png           # Project favicon/logo ```

## 🚀 Getting Started

To run this project locally:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/YourUsername/Animeverse-Gaming.git
    ```
2.  **Navigate to the Directory:**
    ```bash
    cd Animeverse-Gaming
    ```
3.  **Open in Browser:**
    Open the `index.html` file directly in your web browser (e.g., `file:///path/to/Animeverse-Gaming/index.html`).

> **Note on Contact Form:** The contact form uses EmailJS. To make it functional, you will need to replace the placeholder keys in `script.js` with your own EmailJS Public Key, Service ID, and Template ID.
